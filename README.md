# Auto Widgets

A repository to help build faster application in Flutter by providing you customizable widgets.

## Getting Started

Find the Required widget, copy paste it in your code.
 
Widgets that are included :
- Basic Widgets
    - Icon
    - IconButton
    - Image
    - Text
- Layout Widgets
    - Align
    - Box
- Form Widgets
    - SingleUserInput
    - TextControl
    - TextInputDecoration
    - KeyboardType
